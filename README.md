🚀 Python Automation Scripts<br/>
Repositório contendo scripts em Python para automação de tarefas operacionais, manutenção de ambientes e rotinas administrativas.

📌 Objetivo<br/>
Centralizar scripts reutilizáveis para:

🧹 Limpeza automática de arquivos (logs, temporários, uploads)<br/>
📊 Rotinas de monitoramento<br/>
🔄 Automação de processos operacionais<br/>
⚙️ Scripts de suporte para infraestrutura (IIS, serviços, arquivos)<br/>
🛠️ Ferramentas auxiliares para troubleshooting<br/>

⚙️ Requisitos<br/>
Python 3.10+<br/>
Windows Server / Linux<br/>
Permissões adequadas de leitura/escrita nos diretórios<br/>

Instale as dependências:<br/>
pip install -r requirements.txt<br/>

🤝 Contribuição<br/>
Fork do projeto<br/>

Commit suas alterações<br/>
